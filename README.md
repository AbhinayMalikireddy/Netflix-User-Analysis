# Netflix-User-Analysis
## Project Background
Initialy Netflix is website, as a DVD rental service in 1998, later on it started streaming in 2007 and they lanuched Netflix globaly in 2016 via website and mobile app.

This project explores the key reasons behind Netflix’s declining user base. It analyzes trends such as rising subscription costs, increasing competition from other streaming platforms, limited regional content, and changes in user viewing preferences. By examining customer behavior, market shifts, and platform performance, the project aims to identify the major factors contributing to user churn and provide insights that can help improve retention strategies.

## Executive Summary 

#### Overview Findings

This dashboard provides a comprehensive analysis of Netflix’s user performance across multiple years, highlighting trends in revenue, subscription behavior, viewing patterns, and genre preferences. It shows that revenue peaked in 2023 and 2024, while 2022 recorded the lowest performance. The subscription analysis reveals strong user engagement in Premium and Standard plans during the peak years. The viewing pattern chart indicates that evenings are the most active time for movie recommendations, suggesting optimal engagement windows. Additionally, the genre analysis identifies high-potential categories such as Thriller, Sports, and Mystery, offering opportunities for strategic content expansion.
